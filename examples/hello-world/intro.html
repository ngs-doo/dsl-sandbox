Here you can learn about and try out various features of the platform. On the left menu, you'll find a list of examples. Each example provides a short description of one platform concept and some working code which goes along with. You can directly edit and execute PHP code, and interact with the resulting output.

DSL code is read-only. If you'd like to change it, you can register and create a project where you can make changes to DSL and the domain model.

Source code is hosted on Github.
