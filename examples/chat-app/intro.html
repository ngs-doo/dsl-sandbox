Server can push data to client using the SignalR library which fallbacks to appropriate communication mechanism depending on browser.

This example shows client (browser) listening on Chat.Message root. Client is notified each time a new message is created. (hint: open/Run in two separate windows)

This snippet shows enabling notifications in javascript:

    <script src="path/to/jquery.signalR-version.min.js" type="text/javascript"></script>
    <script src="[project-url]/signalr/hubs" type="text/javascript"></script>

    $.connection.hub.start().done(function(){
        $.connection.notifyHub.server.listen('Chat.Message') });
    $.connection.hub.url = '[project-url]'+'signalr/hubs';
    $.connection.notifyHub.client.notify = function(root, uris) {
        console.log('Change on '+root+' with URI(s) '+uris.join(',')) };
